Gateway Script

Ewon Flexy BASIC script for timed WaterlyConnect posts.

The Ewon Flexy integration is a device-side BASIC script intended for Flexy 205 deployments that need to post tag values to WaterlyConnect on a timer. It is a direct path from edge gateway tag collection to Waterly ingestion when the script can own the scheduling logic locally.

Download BASIC script Source folder
205 target Flexy device family
5 min default timer cadence
BASIC copy/paste deployment artifact
Best Fit
Edge gateways that can schedule and submit directly from the Flexy web interface.
Prerequisites

Before loading the script

  • An Ewon Flexy 205 with internet connectivity.
  • Access to the Flexy web interface and BASIC IDE.
  • Waterly token, device ID, and submission URL values.
  • A plan for protecting credentials on the device.
Quick Start

Implementation flow

  1. 1
    Paste the script into BASIC IDE.
    Load the downloaded file into the Flexy web interface and save it.
  2. 2
    Replace the placeholders.
    Set the token, device ID, and Waterly submission URL at the top of the script.
  3. 3
    Run and monitor.
    Enable autorun if needed and verify request status in the Flexy diagnostics.
Configuration Snippet

Credential placeholders

$token$ = "<REDACTED>"
$deviceId$ = "<REDACTED>"
$url$ = "https://app.waterlyapp.com/connect/submit"
