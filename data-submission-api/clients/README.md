# WaterlyConnect Clients

This directory contains the imported WaterlyConnect client libraries and
integration examples that now live alongside the canonical WaterlyConnect
documentation.

The official service contract is maintained in
[`data-submission-api/waterly-connect.yml`](../waterly-connect.yml) and is
published as HTML at
<https://waterlyapp.github.io/waterlyconnect-docs/data-submission-api/>.
